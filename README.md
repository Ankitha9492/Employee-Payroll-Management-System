# Employee-Payroll-Management-System

The Employee Payroll Management System is a Spring Boot web application designed to simplify and automate employee salary management processes. It provides functionalities to add, update, delete, and view employee payroll details efficiently through a user-friendly interface.

# Tech Stack
Category                    Technologies Used
Backend                     Spring Boot, Spring MVC, Spring Data JPA
Frontend                    HTML,CSS, Thymeleaf
Database                    MySQL
IDE                         Eclipse IDE(Enterprise Edition)
Server                      Apache Tomcat(Internal)

# Key Features
-> Add, update, and delete employee payroll details.
-> Manage employee salary, and designation details.
-> View payroll summary in tabular format.
-> Secure database connection using JPA and MySQL.
->RESTful API integration for backend operations.
